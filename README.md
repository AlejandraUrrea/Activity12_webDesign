## Description
This project is an evolution of Activity 12. In this version (Task 10), the application is modularized using **Vue.js Components** and styled with **Bootstrap 5** to create a responsive card-based layout.

## Features
- **Vue Instance:** A new instance that returns an array of Chuck Norris jokes.
- **Dynamic List:** Use of `v-for` to render each joke automatically.
- **Custom Styling:** A modern CSS stylesheet for a better user interface.
