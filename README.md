## Description
This project is a simple web application built with **Vue.js 3**. The goal of this activity is to demonstrate how to create a Vue instance and display data dynamically from an array using the `v-for` directive.

## Features
- **Vue Instance:** A new instance that returns an array of Chuck Norris jokes.
- **Dynamic List:** Use of `v-for` to render each joke automatically.
- **Custom Styling:** A modern CSS stylesheet for a better user interface.
